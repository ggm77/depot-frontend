# Depot

파일을 서버로 전송하는 웹 업로더.

**개발 기간:** 2026.05.25 ~ 2026.05.26

## 기능

- **드래그 앤 드롭** — 파일 또는 폴더를 끌어다 놓아 업로드
- **폴더 자동 압축** — 폴더 선택 시 브라우저에서 ZIP으로 압축 후 전송
- **붙여넣기** — `⌘V` / `Ctrl+V` 로 클립보드 파일 추가
- **업로드 큐** — 파일별 진행률, 속도, 남은 시간 표시
- **일시정지 / 재개 / 재시도** — 항목별 개별 제어
- **동시 업로드** — 최대 1~6개 병렬 전송 (기본값 3)
- **비밀번호 인증** — 업로드 시작 전 비밀번호 필수 입력

## 시작하기

```bash
npm install
npm run dev
```

빌드:

```bash
npm run build
```

## 기술 스택

| 역할 | 라이브러리 |
|------|-----------|
| UI | React 19 + TypeScript |
| 번들러 | Vite 8 |
| ZIP 압축 | JSZip |

## API

업로드 엔드포인트: `POST https://depot.seohamin.com/api/v1/files`

| 필드 | 설명 |
|------|------|
| `files` | 업로드할 파일 (multipart/form-data) |
| `password` | 업로드 비밀번호 |

| 상태 코드 | 의미 |
|-----------|------|
| `204` | 업로드 성공 |
| `400` | 잘못된 요청 (파일 문제) |
| `401` | 비밀번호 불일치 |
